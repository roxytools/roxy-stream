# Security Policy

## Supported Versions

| Operating System | Status         |
| ---------------- | -------------- |
| Windows          | Fully Working  |
| Linux            | Not Yet Tested |

If you have successfully tested one of the unverified platforms, please send a video demonstration via Discord (username: stolenbmws). Verified test results will be used to update this table.

## Reporting a Vulnerability

If you discover a vulnerability, bug, or security concern, you may report it in one of the following ways:

1. Submit an issue through the Issues section:
   `../../issues`

2. Contact the developer directly through the support link:
   `../../?tab=readme-ov-file#issues--support`

3. For severe or sensitive security issues, you may report them privately via [email](mailto:owen@owendev.cfd):

All legitimate reports will be reviewed and handled as quickly as possible.

## Virus and Malware Statement

This project has been tested and scanned through multiple antivirus and malware detection tools. It contains zero viruses or malicious code. The source is open for review, and users are encouraged to run their own scans if needed. This is a hobby project and is maintained with transparency and safety in mind.
